# Digital Timezone Clock (8086 Assembly Language)

## COAL Lab Final Project

Developed a multi-timezone digital clock using 8086 Assembly Language.

### Features

- UTC Time Display
- Pakistan Time (UTC+5)
- USA Time (UTC-5)
- Real-Time Clock
- Overflow Handling
- Continuous Execution

### Concepts Used

- Registers
- Counters
- Loops
- Conditional Jumps
- DOS Interrupts
- Time Zone Conversion

### Technologies

- 8086 Assembly Language
- MASM
- DOS Interrupts
